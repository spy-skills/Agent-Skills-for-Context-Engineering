# Project Context Index — 05_context-engineering-skills

Folder này lưu trữ learnings, decisions, và context quan trọng của dự án.
Claude sessions sau sẽ đọc folder này để hiểu bối cảnh.

## Dự án
Phân tích repo "Agent Skills for Context Engineering" — bộ sưu tập 13 skills về context engineering cho AI agents. Mục tiêu: nghiên cứu, đánh giá chất lượng, xếp hạng ưu tiên, và tạo lộ trình học tập.

## Cấu trúc
- `decisions/` — Quyết định quan trọng
- `learnings/` — Kiến thức học được
- `context/` — Bối cảnh dự án
- `daily/` — Nhật ký hàng ngày
- `tasks/` — Context cho từng task
- `leverage/` — Phân tích đòn bẩy
- `samples/` — Mẫu công việc

## Cách sử dụng
- `/viec ghi` — Ghi chép mới
- `/viec doc` — Đọc lại context
- `/viec xong` — Tự động hỏi learnings khi hoàn thành task
- `/viec review` — Tổng hợp vào daily log
